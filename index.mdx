Polymarket Data is a comprehensive dataset and open-source toolkit for Polymarket prediction markets. It provides 1.1 billion on-chain trading records fetched directly from the Polygon blockchain, processed into five analysis-ready Parquet formats covering raw events, trades, market metadata, quantitative analysis, and user behavior.

## What's in the dataset

The dataset covers every OrderFilled event from Polymarket's two official exchange contracts on Polygon, from inception through early 2026. Five processed files serve different research needs:

| File | Size | Records | Best for |
|------|------|---------|----------|
| `orderfilled.parquet` | 31 GB | 293.3M | Raw blockchain event analysis |
| `trades.parquet` | 32 GB | 293.3M | Trade-level analysis with market metadata |
| `markets.parquet` | 68 MB | 268,706 | Market lookup and metadata joins |
| `quant.parquet` | 21 GB | 170.3M | Price studies, time-series forecasting |
| `users.parquet` | 23 GB | 340.6M | PnL calculation, wallet profiling |

## Get started

<Steps>
  <Step title="Download the dataset">
    Get all historical data from HuggingFace — no setup required.

    ```bash
    pip install huggingface_hub
    hf download SII-WANGZJ/Polymarket_data --repo-type dataset
    ```
  </Step>
  <Step title="Install the toolkit">
    Install the Python package to fetch fresh data or run the full pipeline.

    ```bash
    git clone https://github.com/SII-WANGZJ/Polymarket_data.git
    cd Polymarket_data
    pip install -e .
    ```
  </Step>
  <Step title="Analyze the data">
    Load any Parquet file with pandas and start exploring.

    ```python
    import pandas as pd

    df = pd.read_parquet('quant.parquet')
    market_stats = df.groupby('market_id').agg({
        'usd_amount': 'sum',
        'price': ['mean', 'std'],
        'transaction_hash': 'count'
    })
    print(market_stats.head())
    ```
  </Step>
  <Step title="Fetch live data">
    Run continuous mode to keep your local copy synced with the blockchain.

    ```bash
    ./scripts/continuous_start.sh
    ```
  </Step>
</Steps>
